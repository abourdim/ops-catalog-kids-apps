# Pattern Maker

**Create beautiful geometric patterns** — an interactive, canvas-based geometric pattern generator for kids, inspired by Islamic geometry.

## Features

- **5 Shapes**: Triangle, Square, Hexagon, Circle, Star
- **8 Color Palette**: Vibrant kid-friendly colors with alternating pattern support
- **3 Grid Sizes**: 4x4, 6x6, 8x8
- **Rotation**: 0°, 45°, 90° per-cell rotation
- **Mirror Toggle**: Flip shapes across axes for kaleidoscope effects
- **5 Pre-made Templates**: Honeycomb, Checkerboard, Star Grid, Diamond, Zigzag
- **Random Generator**: One-click surprise patterns
- **Save as PNG**: Download your creation
- **Activity Log**: Tracks all pattern changes

## Controls

| Control       | Description                          |
|---------------|--------------------------------------|
| Shape selector| Triangle, Square, Hexagon, Circle, Star |
| Color palette | 8 colors — click to select           |
| Grid size     | 4x4, 6x6, 8x8                       |
| Rotation      | 0°, 45°, 90°                        |
| Mirror        | Toggle horizontal/vertical mirroring |
| Templates     | 5 pre-made patterns                  |
| Random        | Generate a random pattern            |
| Save PNG      | Download pattern as image            |

## Keyboard Shortcuts

| Key     | Action              |
|---------|---------------------|
| Enter   | Start creating      |
| 1-5     | Select shape        |
| R       | Random pattern      |
| S       | Save PNG            |
| M       | Toggle mirror       |
| Escape  | Close help panel    |

## Themes

8 built-in themes: Terminal, Midnight, Ember, Phosphor, Signal, Redshift, Arctic, Sand.

## Languages

Trilingual: English, Français, العربية (with RTL support).

## Technical

- Single HTML file, zero dependencies, fully offline
- Canvas-based real-time rendering
- Web Audio API for sound feedback
- localStorage for persisting preferences (theme, language, sound, high score)
- HS_KEY: `wdiy-hs-pattern-maker`

## Privacy

Local-first. No tracking, no analytics, no external calls. All data stays in the browser.
