❓ Help
What is Sudoku?

Fill the grid. Every row, column, and box — each number exactly once.

Does it work offline?

Yes. Once loaded, everything runs in your browser with no internet needed.

Can I play on my phone?

Yes. Works on any device with a modern browser — phone, tablet, laptop, desktop.

How do I change the theme?

Click any theme button (🖥 🌌 🔥 etc.) in the controls bar.

How do I change the language?

Click EN / FR / ع. Arabic enables right-to-left layout automatically.

Is my data private?

Yes. Everything runs locally in your browser. No data is sent anywhere.

🎯 Purpose

Fill the grid. Every row, column, and box — each number exactly once.

📚 What You'll Learn

Logic and deduction, number placement strategy, process of elimination, patience and focus.

🎨 Themes

8 built-in themes: Terminal, Midnight, Ember, Phosphor, Signal, Redshift, Arctic, Sand.

🌐 Languages

Trilingual: English, Français, العربية. Arabic auto-enables RTL layout.

🔊 Sound

Web Audio API — no files needed. Correct = rising tone, wrong = buzz, win = melody. Toggle with 🔊 button.

🔗 Based On

Open-source educational software. See README for credits.

🔒 Privacy

Local-first. No tracking, no analytics, no external calls. All data stays in your browser.
