# World Map Quiz

Interactive canvas-based world map quiz for kids. Click on continents and countries to test your geography knowledge.

## Features

- **Canvas-drawn world map** with simplified but recognizable continent outlines
- **3 difficulty levels**: Easy (7 continents), Medium (continents + 10 countries), Hard (20 countries)
- **20 countries**: Egypt, Morocco, Turkey, Saudi Arabia, Pakistan, Indonesia, Malaysia, India, China, Japan, France, UK, Germany, USA, Canada, Brazil, Australia, South Africa, Nigeria, Russia
- **Info popups** on correct answers showing capital, population, and fun facts
- **Wrong answer feedback** with red flash and correct location highlighted
- **Progress bar, timer, score tracking** with persistent high score
- **8 CSS themes**: Terminal, Midnight, Ember, Arctic, Phosphor, Signal, Redshift, Sand
- **3 languages**: English, French, Arabic (RTL support)
- **Sound effects** via Web Audio API (no files needed)
- **Sidebar help** with FAQ, How-To, and Wiki tabs
- **Keyboard shortcuts**: Enter (start), H (help), N (new game), Escape (close help)
- **Offline**: Single HTML file, zero dependencies

## Usage

Open `index.html` in any modern browser. No server required.
