## 🎯 Purpose

The world's most popular logic puzzle. Fill a grid so that every row, every column, and every box contains each number exactly once. Builds systematic logical thinking.

## 📖 What You'll Learn

- Logical deduction — elimination and constraint satisfaction
- Systematic thinking — checking rows, columns, and boxes methodically
- Number sense — working with digits 1-9 (or 1-4 for beginners)
- Patience and persistence — complex puzzles require careful work
- Error detection — spotting conflicts in your solution

## 🎮 How to Play

1. Choose grid size: 4×4 (Easy) or 9×9 (Classic)
2. Choose difficulty: Easy / Medium / Hard
3. Press ▶ New Game
4. Click an empty cell to select it
5. Press a number key (or tap the number pad) to fill it in
6. Use 💡 Hint to reveal a cell if stuck
7. Use ✔ Check to verify your progress
8. Fill all cells correctly to win!

## 📊 Grid Sizes

| Size | Numbers | Boxes | Best For |
|------|---------|-------|----------|
| 4×4 | 1-4 | 2×2 | Ages 5-7, beginners |
| 9×9 | 1-9 | 3×3 | Ages 8+, classic puzzle |

## 💡 Tips

- Start by scanning for cells with only one possible number
- Look for rows/columns that are nearly complete
- In 9×9, check each 3×3 box for missing numbers
- Use the Check button to catch errors early

## 🔗 Based On

**GCompris Sudoku** — the puzzle was popularized in 1986 by Japanese publisher Nikoli, based on "Number Place" by Howard Garns (1979).
