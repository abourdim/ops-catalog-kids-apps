# Workshop-DIY — Kids Apps

**190 interactive educational activities for kids. Real games, not quizzes.**

GCompris, TuxMath, TuxType recreations as single HTML files.

## Features

- 8 visual themes
- 3 languages (EN, FR, AR with RTL)
- Offline-capable
- Zero dependencies
- Canvas games, typed input, drag-and-drop, custom UI
