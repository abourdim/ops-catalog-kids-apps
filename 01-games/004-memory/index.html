❓ Help
What is this game?

A card-matching memory game. Flip two cards at a time to find matching pairs. Clear all pairs to win.

What are the game modes?

Emoji: match identical pairs. Letters: match uppercase to lowercase. Math: match operations to results (3×2 ↔ 6).

What grid size should I start with?

Ages 2-3: start with 2×2. Ages 4-6: try 3×4. Ages 8+: challenge yourself with 4×6.

How is scoring measured?

By number of moves (each flip of 2 cards = 1 move). Fewer moves = stronger memory.

What's a perfect game?

When every move is a match: moves = number of pairs. On a 4×4 grid (8 pairs), perfect = 8 moves.

Does Math mode teach math?

Yes! You must calculate 3×2 to know it matches the card showing 6. Memory + arithmetic combined.

🎯 Learning Goals

Visual memory, concentration, strategic thinking, pattern recognition, letter case awareness (letter mode), arithmetic (math mode).

📊 Benchmarks

3×4 grid: Good=12 moves, Great=8, Perfect=6. 4×4 grid: Good=18, Great=12, Perfect=8.

🎮 Game Modes

Emoji: identical pairs (🚀↔🚀). Letters: uppercase↔lowercase (A↔a). Math: operation↔result (8÷2↔4).

🔗 Based On

GCompris Memory by Bruno Coudoin (2000-present). Original includes image, sound, math, and letter variants. gcompris.net

💡 Pro Tips

Scan the whole board first. Use a strategy: flip systematically row by row. When you see a non-match, remember both positions for later.
