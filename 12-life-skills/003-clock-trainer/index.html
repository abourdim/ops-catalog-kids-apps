❓ Help
What is Clock Trainer?

An interactive game to learn how to read analog clocks and tell time. Two modes: Quiz (read the clock) and Set (move the hands).

Does it work offline?

Yes. Once loaded, everything runs in your browser with no internet needed.

Can I play on my phone?

Yes. Works on any device with a modern browser — phone, tablet, laptop, desktop. In Set mode, drag the hands with your finger.

How do I change the theme?

Click any theme button in the controls bar.

What are the difficulty levels?

Easy = hours only (:00, 8 questions). Medium = adds :15, :30, :45 (12 questions). Hard = any 5-minute interval (15 questions).

Is my data private?

Yes. Everything runs locally in your browser. No data is sent anywhere.

Purpose

Teach kids to read analog clocks. Quiz mode tests reading, Set mode tests hand placement.

Themes

8 built-in themes: Terminal, Midnight, Ember, Phosphor, Signal, Redshift, Arctic, Sand.

Sound

Web Audio API — no files needed. Correct = rising tone, wrong = buzz, win = melody. Toggle with the sound button.

Daily Activities

After each correct answer, a fun daily activity hint is shown for that time (e.g., "7:00 AM — Wake up time!").

Privacy

Local-first. No tracking, no analytics, no external calls. All data stays in your browser.
