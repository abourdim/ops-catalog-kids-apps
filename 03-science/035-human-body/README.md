# Human Body Explorer

Interactive canvas-based human body diagram for kids. Click organs to learn or take a quiz.

## Features

- **Learn Mode**: Click any organ to see its name, function, and a fun fact
- **Quiz Mode**: "Click on the [organ]!" — find the correct location on the body
- **Difficulty levels**: Easy (5 major organs), Medium (8 organs), Hard (10+ organs)
- **Canvas drawing**: Simple human body outline with clickable organ hotspots
- **Organs**: Brain, Heart, Lungs, Stomach, Liver, Kidneys, Intestines, Bones, Muscles, Skin, Eyes
- **Trilingual**: English, French, Arabic (with RTL support)
- **8 CSS themes**: Terminal, Midnight, Ember, Arctic, Phosphor, Signal, Redshift, Sand
- **Score tracking** with localStorage high score persistence
- **Sound effects** via Web Audio API (no external files)
- **Keyboard shortcuts**: Enter (start), Q (quiz mode), L (learn mode), Escape (close help)
- **Splash screen** with Bismillah, Start Mission, lang/theme switchers
- **Sidebar help** with FAQ, How-To, and Wiki tabs
- **Single HTML file**, zero dependencies, fully offline

## Storage

- `wdiy-hs-human-body` — high score (percentage)
- `wdiy-lang` — language preference
- `wdiy-theme` — theme preference
- `wdiy-mute` — sound mute state

## Usage

Open `index.html` in any modern browser. No server required.
